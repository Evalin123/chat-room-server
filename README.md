# Chat Room

A simple chat room with NodeJS and Socket.io.

## Technologies
Project is created with:
* JavaScript
* [Node JS](https://nodejs.org/en/)
* [Socket.io](https://socket.io/)
* [Express](https://expressjs.com/)
* [cors](https://www.npmjs.com/package/cors)

## Setup

```
$npm install
$npm start
```
